# deloitte-telemetry-downtime-dashboard
📊 Factory Downtime Analysis – Tableau Dashboard 📌 Project Overview  This project analyzes telemetry data to identify operational downtime patterns across factories and device types. The goal was to quantify potential downtime and highlight risk areas using data visualization.

🛠 Tools Used

Tableau

Excel

JSON telemetry dataset

🔎 Methodology

Imported and structured JSON telemetry data in Tableau

Created a calculated field to assign 10 minutes of downtime per unhealthy status

Built visualizations:

Down Time per Factory

Down Time per Device Type

Used Excel to classify equity scores into business-defined categories

Derived operational insights from aggregated downtime metrics

📈 Key Insights

Identified factories with the highest downtime exposure

Determined device types contributing most to operational risk

Highlighted patterns that could inform maintenance prioritization
